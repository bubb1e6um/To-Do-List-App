#  To-Do List App

Современный и минималистичный менеджер задач с поддержкой темной темы и генератором идей на базе API.

## Описание

Это веб-приложение помогает организовать день, записывать важные дела и фильтровать их. 
Главная фишка — **интеграция с внешним API**. Если вам скучно или вы не знаете, чем заняться, нажмите на "Волшебную палочку" , и приложение само предложит вам интересную задачу!

##  Функционал

- **CRUD операции:** Создание, чтение, редактирование и удаление задач.
- ** Генератор идей (API):** Получение случайных задач с сервера при нажатии на кнопку магии.
- ** Темная тема:** Полноценная поддержка Dark Mode с сохранением комфорта для глаз.
- ** Поиск и Фильтр:** Мгновенный поиск по тексту и фильтрация (Все / Выполненные / Невыполненные).
- ** Адаптивность:** Удобно использовать как на ПК, так и на смартфоне.
- **Floating Action Button:** Красивая плавающая кнопка для добавления заметок.

##  Технологии

Проект создан на чистом стеке (Vanilla JS) без использования тяжелых фреймворков:

- **HTML5** — Семантическая верстка.
- **CSS3** — Flexbox, CSS Variables (для темной темы), анимации.
- **JavaScript (ES6+)** — Асинхронные запросы (`async/await`), манипуляции с DOM.

##  API Reference

В проекте используется бесплатный API для генерации случайных дел:

| API | Endpoint | Описание |
| :--- | :--- | :--- |
| **DummyJSON** | `https://dummyjson.com/todos/random` | Возвращает случайную задачу (Todo) в формате JSON. |
